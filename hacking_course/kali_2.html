Quick Recap and What's Next

In our previous tutorial, we learned what Kali Linux is and how to install it. Now, we will dive into the heart of Linux: the command line terminal. We'll explore some fundamental commands you will use daily.

Basic Kali Linux Commands

The terminal is a powerful tool. Let's get comfortable with it by learning these essential commands. Below are the commands with explanations and examples.

  • whoami
  • pwd
  • ls

    ls -l
  • cd

    cd Documents or cd ..
  • cat

    cat filename.txt
  • touch

    touch newfile.txt
  • mkdir

    mkdir new_folder
  • echo

    echo "Hello World" > hello.txt
  • ifconfig / ip addr
  • iwconfig
  • apt

    apt update or apt install packagename
  • ping

    ping google.com
  • history
  • clear
  • command --help

    ls --help
  • uname

    uname -a
  • neofetch
  • date
